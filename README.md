## ğŸ§  How It Works

1. Movie data is loaded from a CSV file
2. Genres and descriptions are combined
3. TF-IDF Vectorizer converts text into numerical form
4. Cosine Similarity finds similar movies
5. Flask displays recommendations on the web interface
6. TMDB API fetches real-time movie posters
